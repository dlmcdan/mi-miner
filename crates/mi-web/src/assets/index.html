Option 1: Generate a New Wallet

We'll create a BIP39 wallet with a 12-word recovery phrase and a bc1q address. No external software needed. Choose a passphrase to encrypt the phrase on disk.

Option 2: Use an Existing Address

Already have a wallet (Sparrow, Electrum, Ledger, Coinbase, etc.)? Paste your Bitcoin receive address below.
